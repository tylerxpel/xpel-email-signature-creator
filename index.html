How To Use

Copy your signature, then follow the steps below to install it in Outlook.

Copies the formatted signature — images, layout, and all. Paste directly into the Outlook editor.

Outlook Web & New Desktop App

outlook.com · Microsoft 365 · New Outlook for Windows / Mac

  1. 1 Click the Settings gear icon in the top-right corner of Outlook.
  2. 2 Navigate to Mail → Compose and reply → Email signature.
  3. 3 Click New signature and give it a name (e.g. XPEL).
  4. 4 Click into the signature editor, then press Ctrl+V (or ⌘V on Mac) to paste.
  5. 5 Click Save. Set it as your default for new messages and/or replies using the dropdowns above the editor.

Classic Outlook Desktop

Outlook 2016 · 2019 · 2021 (Windows)

  1. 1 Open Outlook and go to File → Options → Mail.
  2. 2 Click Signatures… under the Compose messages section.
  3. 3 Click New, enter a name (e.g. XPEL), and click OK.
  4. 4 Click into the editor area at the bottom, then press Ctrl+V to paste.
  5. 5 Click OK to save. Use the dropdowns at top right to set it as default for New messages and/or Replies/forwards.
